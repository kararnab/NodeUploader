# Upload Server
## Description
Demonstrating multiple ways to file upload.

## Requirements

* [node.js](http://nodejs.org/) -- v18.12.0 or newer

## Setup
```
yarn install
yarn start
```

### Steps: 
- Open web browser and go to http://localhost:2000
- Click choose files
- Press upload button
- Files will be uploaded and stored in `uploadedFiles` folder
